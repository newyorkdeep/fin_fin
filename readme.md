To run this app you need to first create a file called .env in "server" folder.
In this file you need to add the line:
`API_KEY=sdlk89dsfh29...`
But instead of this giberish you need to paste your actual API key that you have
to get from ExchangeRate api website. 
Then you need to run npm `run install-all` 
And then `npm run dev`
